# Brainstorm de Design - Guia de Solicitações Acadêmicas

## Abordagem Escolhida: Modernismo Acadêmico com Acessibilidade Elevada

### Design Movement
**Modernismo Acadêmico** - Uma fusão entre a elegância institucional tradicional e o minimalismo contemporâneo. Inspirado em interfaces educacionais de alto padrão que priorizam clareza, hierarquia visual e confiança.

### Core Principles
1. **Clareza Hierárquica**: Cada serviço acadêmico é apresentado com prioridade visual clara, facilitando a navegação para alunos com diferentes níveis de familiaridade com processos administrativos.
2. **Confiança Institucional**: Uso de cores sóbrias e tipografia profissional que transmitem autoridade e segurança, essencial para um guia oficial de procedimentos.
3. **Acessibilidade Proativa**: Contraste adequado, espaçamento generoso e tamanhos de fonte legíveis para garantir que todos os alunos possam acessar as informações.
4. **Fluidez Interativa**: Transições suaves e feedback visual imediato que tornam a exploração dos serviços intuitiva e agradável.

### Color Philosophy
- **Primária**: Azul profundo (`#1e3a8a`) - Transmite confiança, profissionalismo e segurança. Associado com instituições educacionais.
- **Secundária**: Dourado suave (`#d4af37`) - Acentos elegantes que destacam elementos importantes sem ser agressivo.
- **Neutros**: Branco puro e cinza claro para fundos e textos, garantindo legibilidade máxima.
- **Avisos**: Âmbar/laranja suave para alertas sobre prazos, mantendo a harmonia visual.

**Reasoning**: A combinação azul + dourado é clássica em ambientes acadêmicos, criando uma sensação de prestígio e confiabilidade. O branco maximiza a legibilidade e cria espaço respirável.

### Layout Paradigm
- **Hero Section**: Banner expansivo com gradiente sutil (azul profundo → azul médio) e uma ilustração abstrata que representa "jornada acadêmica".
- **Card Grid**: Serviços organizados em grid responsivo (3 colunas no desktop, 1-2 no mobile) com cards que possuem bordas sutis e sombras delicadas.
- **Sidebar Informativo**: Painel lateral com dicas rápidas, prazos importantes e links de suporte (em desktop; colapsível em mobile).
- **Footer Institucional**: Rodapé com contatos, links úteis e marca da instituição.

### Signature Elements
1. **Ícones Personalizados**: Cada serviço tem um ícone único (ex: documento para emissão, calendário para renovação) que facilita o reconhecimento visual rápido.
2. **Badges de Status**: Pequenos badges indicam se um serviço tem prazos urgentes, documentação necessária ou informações especiais.
3. **Linhas Decorativas**: Linhas horizontais sutis em dourado que separam seções, criando ritmo visual sem poluição.

### Interaction Philosophy
- **Hover States**: Cards ganham uma sombra mais pronunciada e levemente se elevam ao passar o mouse, criando feedback tátil.
- **Expandable Details**: Cada card pode ser expandido para revelar mais detalhes, mantendo a página limpa inicialmente.
- **Tooltips Informativos**: Hover em ícones revela explicações rápidas sobre cada serviço.
- **Smooth Scrolling**: Navegação fluida entre seções com scroll suave.

### Animation
- **Entrance**: Cards aparecem com fade-in suave + slide-up leve (100ms de duração) quando a página carrega, criando dinamismo sem distração.
- **Hover**: Transição suave (200ms) para sombra e elevação ao passar o mouse.
- **Expand/Collapse**: Animação de altura suave (300ms) ao expandir cards para mostrar detalhes.
- **Scroll Trigger**: Elementos ganham um efeito de fade-in leve conforme o usuário faz scroll, mantendo o engajamento.

### Typography System
- **Display Font**: `Playfair Display` (700) para títulos principais - elegante, profissional, memorável.
- **Heading Font**: `Montserrat` (600) para subtítulos e títulos de cards - moderna, legível, hierárquica.
- **Body Font**: `Inter` (400/500) para textos - altamente legível, neutra, profissional.
- **Hierarchy**:
  - H1 (Hero): 48px, Playfair Display 700
  - H2 (Seção): 32px, Montserrat 600
  - H3 (Card): 24px, Montserrat 600
  - Body: 16px, Inter 400
  - Small: 14px, Inter 400

### Design Rationale
Este design foi escolhido porque:
1. **Confiança**: Cores e tipografia transmitem profissionalismo essencial para um guia oficial.
2. **Acessibilidade**: Alto contraste, espaçamento generoso e fontes legíveis garantem inclusão.
3. **Modernidade**: Evita parecer desatualizado, mas mantém a sobriedade apropriada para contexto acadêmico.
4. **Escalabilidade**: Layout flexível funciona bem em todos os dispositivos sem perder identidade visual.
5. **Engajamento**: Animações sutis e interatividade mantêm os usuários engajados sem ser distrativo.
